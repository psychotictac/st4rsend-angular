    Description

    Cette application est un essai de documentation pour étude des technologies ci-dessous:
  • La présentation client en mode Single Page Application (SPA) s'appuie sur le framework Angular 2+.
  • Les services cloud Google Compute Engine (GCE) fournissent l'infrastructure de production qui héberge les services servers www ansi que l'espace de développement.
  • Le système d'exploitation Debian anime les serveurs de production (DataCenter side).
  • Côté "Office/Bureau" le système d'exploitation Windows 10 héberge le Subsystem Linux (WSL) pour accès SSH distant aux environnements de développement (WSL Terminal + VIM), ainsi que les navigateurs webs pour la visualisation.
  • Le serveur web NGINX couplé aux services de load balancing front pour groupe d'instances GCE.
  • La base de donnée MariaDB.
  • Le language GO pour le développement côté server (backend).
  • Un protocol de communication propriétaire s'appuyant sur la technologie websocket (RFC 6455).
